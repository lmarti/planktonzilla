## Platform Requirements

**Development:**
- Linux container (Ubuntu 22.04 inside dev container) or any Python 3.11–3.13 host with Poetry.
- macOS host supported for editing (current working tree is on Darwin) but training requires CUDA GPUs.
- VS Code Dev Container (Docker required, Docker socket mounted).

**Production / training:**
- NVIDIA GPU with CUDA 12.5 driver — H100 in production (Jean Zay `gpu_p6` partition, `qos_gpu_h100-t3`).
- Multi-node distributed training via `torchrun` + SLURM (8–16 nodes × 4 H100s in `scripts/train.sh`, `scripts/train_clip.sh`).
- Disk: large enough for HF dataset cache and `data/` (e.g. `data/isiisnetdatasetimporter_imagefolder/` already has 32 class folders).
