# Important information

This folder contains public dataset that are stored here only for technical reasons.

## Plankton lensless images dataset

Pastore, V.P., Zimmerman, T.G., Biswas, S.K. et al. Annotation-free learning of plankton for classification and anomaly detection. Sci Rep 10, 12142 (2020). <https://doi.org/10.1038/s41598-020-68662-3>

Original dataset is available online at <https://ibm.ent.box.com/v/PlanktonData>.

Local dataset: `lensless_dataset.zip`.

Modifications of original dataset:

* contains only the images organized by labels, and
* posted it online accessible in a programmatic way via Python (original dataset is available via box.com).
